# TaskFlow Hub 📋

A responsive web app with Contact Form and To-Do List using HTML, CSS and JavaScript.

## Features

### Contact Form
- Opens by default
- Form validation
- Invalid email error
- Success message

### To-Do List
- Add tasks
- Delete tasks
- Complete tasks

### Responsive
- Works on desktop and mobile

## Technologies

- HTML
- CSS
- JavaScript

## Run

Open 'index.html' in your browser.
